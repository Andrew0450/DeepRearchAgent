# 管线调研Agent

基于LangGraph工作流构建的智能调研Agent，严格执行调研SOP v3.1，提供高质量、结构化、可追溯的专业调研服务。

## 核心能力

### 1. 智能方向识别
根据用户输入自动识别6种调研方向：

| 方向 | 类型 | 适用场景 |
|------|------|----------|
| **A** | 市场规模与增长趋势 | 了解行业/产品整体规模、增速、预测 |
| **B1** | 行业驱动型调研 | 产业链分析、竞争格局、头部玩家、行业趋势 |
| **B2** | 产品驱动型调研 | 产品市场细分、STP分析、应用场景、产业链 |
| **C** | 技术路线与专利布局 | 技术演进、专利分析、创新趋势 |
| **D** | 竞品/产品分析 | 功能对比、性能测试、成本分析、方案选型 |
| **E** | 政策与趋势 | 政策解读、监管动态、影响分析、合规风险 |

### 2. 严格工作流执行
采用LangGraph StateGraph架构，按SOP规定的顺序执行：
```
用户输入 → 提取主题 → 识别方向 → B1方向调研 → E方向调研 → B2+D方向调研 → A/C方向调研 → 整合报告 → 生成摘要 → 输出DOCX
```

### 3. 权威数据来源
每个方向搜索时指定权威来源，确保数据可信度：
- **行业数据**：Wind、Statista、Gartner、IDC、艾瑞咨询、赛迪顾问
- **政策文件**：中国政府网、国务院官网、发改委、财政部、各部委官网
- **技术文献**：IEEE、ACM、Nature、arXiv、Google Scholar
- **企业信息**：企业年报、官网、权威财经媒体

### 4. 报告生成
- 输出格式：Markdown摘要 + DOCX完整报告
- 数据来源：每条数据标注具体URL
- 时效标注：超2年数据标注"[数据可能过时]"
- 估算透明：估算数据附公式+参数来源+假设+⚠️标注

## 快速开始

### 本地运行完整工作流
```bash
bash scripts/local_run.sh -m flow
```

### 运行指定节点
```bash
bash scripts/local_run.sh -m node -n extract_topic
```

可用节点列表：
- `extract_topic` - 提取调研主题
- `identify_directions` - 识别调研方向
- `research_b1` - 行业驱动型调研
- `research_e` - 政策与趋势调研
- `research_b2` - 产品驱动型调研
- `research_d` - 竞品/产品分析
- `research_ac` - 市场规模与技术路线
- `integrate_report` - 整合报告
- `generate_summary` - 生成摘要
- `generate_docx` - 生成DOCX文件

### 启动HTTP服务
```bash
bash scripts/http_run.sh -m http -p 5000
```

## 使用示例

### 示例1：行业调研
```
请调研一下人工智能芯片市场
```
Agent自动识别为B1方向，输出行业定义、产业链、市场规模、竞争格局、头部企业、趋势判断等完整内容。

### 示例2：政策调研
```
请调研一下2024年新能源汽车相关政策
```
Agent自动识别为E方向，输出政策背景、关键人物讲话、政策解读、影响分析、落地进展、风险提示等。

### 示例3：竞品分析
```
帮我分析一下腾讯云和阿里云的竞争情况
```
Agent自动识别为D方向，输出需求定义、候选方案、功能对比矩阵、性能对比、成本对比、优劣势分析等。

## 数据来源规范

### 强制要求
1. **每条数据必须有来源URL**，禁止使用"据统计"、"据报道"等模糊表述
2. **权威来源优先**：政府官网 > 行业协会 > 权威智库 > 上市公司年报 > 专业媒体
3. **数据时效**：超2年数据标注"[数据可能过时]"
4. **估算透明**：估算数据必须附公式+参数来源+假设+⚠️标注
5. **无法溯源标注**：无法找到公开来源的数据标注"未检索到公开数据"

### 各方向推荐来源
- **B1行业调研**：Wind、Statista、Gartner、IDC、艾瑞、赛迪、中商产业研究院
- **B2产品调研**：Gartner、IDC、Forrester、TAM/SAM/SOM分析报告
- **D竞品分析**：Gartner Magic Quadrant、IDC MarketScape、企业年报
- **E政策调研**：中国政府网、国务院、各部委官网、新华社、人民日报
- **A/C技术调研**：IEEE、ACM、Nature、Science、arXiv、专利数据库

## 输出格式

### 最终输出
```markdown
### 报告摘要
[200字以内的核心发现摘要]

### 完整报告下载链接
[报告标题.docx](下载链接)
```

### 完整报告结构（以B1为例）
```
## 1. 行业概述
### 1.1 行业定义与范围
### 1.2 发展阶段判断

## 2. 产业链拆解
### 2.1 上游/中游/下游图谱
### 2.2 各环节价值分布

## 3. 市场规模与增速
### 3.1 历史规模与增速（表格）
### 3.2 未来3-5年预测
### 3.3 数据来源标注

## 4. 行业痛点
### 4.1 核心问题（至少3个）
### 4.2 技术/市场层面瓶颈
### 4.3 未被满足的需求

## 5. 竞争格局
### 5.1 头部玩家与市场份额（表格）
### 5.2 CR3/CR5集中度
### 5.3 竞争要素分析

## 6. 关键玩家（TOP 5-8企业画像）
- 成立时间、核心产品、市场地位、最新数据、官网

## 7. 驱动与抑制因素
### 7.1 技术驱动
### 7.2 需求驱动
### 7.3 抑制因素/风险

## 8. 趋势判断
### 8.1 短期（1年）
### 8.2 中期（3年）
### 8.3 长期（5年+）
```

## 技术栈

- **LangGraph**：工作流编排
- **LangChain**：Agent框架
- **coze-coding-dev-sdk**：工具集成（搜索、文档生成）
- **OpenAI API**：大语言模型调用

## 注意事项

1. 调研结果基于公开网络数据，仅供参考，不构成投资建议
2. 部分估算数据标注了⚠️，使用时请注意
3. 报告生成需要一定时间，请耐心等待
4. 复杂调研主题可能需要2-3分钟完成
